# Customer Churn Prediction System

This project predicts whether a customer is likely to leave a telecom service
based on demographic details, service usage, and billing information.

## Features
- Machine learning–based churn prediction
- Business-style interactive dashboard
- End-to-end pipeline (training → deployment)

## Tech Stack
- Python
- scikit-learn
- Pandas, NumPy
- Streamlit

## Disclaimer
This application is for educational and decision-support purposes only.
